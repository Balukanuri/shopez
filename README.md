# ShopEZ - MERN E-Commerce Platform

## Overview

ShopEZ is a full-stack e-commerce web application built using the MERN stack.

Users can browse products, register/login, add items to cart, place orders, and track previous orders.

Administrators can manage products and customer orders through a dedicated admin dashboard.

---

## Features

### User
- Register/Login
- JWT Authentication
- Browse Products
- Product Details
- Add to Cart
- Remove from Cart
- Place Orders
- Order History

### Admin
- Admin Dashboard
- Add Products
- Update Products
- Delete Products
- View Orders
- Manage Inventory
